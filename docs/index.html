Web Quality Gatekeeper

Automated quality gates for web projects — accessibility, performance, and visual regression in one CLI.

Core Checks

  • Playwright Smoke

    Headless browser screenshots with deterministic rendering.

  • axe Accessibility

    WCAG compliance scanning with severity counts.

  • Lighthouse Perf

    Budget enforcement for score, LCP, CLS, and TBT.

  • Visual Regression

    Pixel-level diff against committed baselines.

  • Security-Aware Target Policy

    Blocks internal/private targets in CI or authenticated runs unless explicitly allowed.

  • Runtime Vulnerability Gate

    CI enforces high/critical runtime dependency audits with expiring exceptions.

Quickstart

npm ci
npx playwright install
npm run build
npx wqg audit https://example.com

See the GitHub repository for full documentation.
